# egg-born-module-cabloy-cmsthemecommunity

Cabloy-CMS Theme

## Installation

``` bash
$ npm i egg-born-module-cabloy-cmsthemecommunity
```

## Usage

Changing languange's theme to `cabloy-cmsthemecommunity`

``` javascript
"language": {
  "default": "en-us",
  "items": "en-us"
},
"themes": {
  "en-us": "cabloy-cmsthemecommunity"
}
```
